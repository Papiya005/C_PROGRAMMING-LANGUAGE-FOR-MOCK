# 💻 C Programming Mock Preparation Repository

## 📌 About

This repository contains a collection of C programming mock questions and solutions designed to help beginners and students strengthen their core programming concepts. It covers important topics frequently asked in exams and coding interviews.

---

## 🎯 Objectives

* Practice C programming concepts
* Improve problem-solving skills
* Prepare for exams and technical interviews
* Build strong fundamentals in C

## 🧪 Sample Problem

```c
#include <stdio.h>
#include <string.h>

void printregion(char* user){
    if(strcmp(user, "Indian") == 0){
        printf("Namaste");
    }
    else if(strcmp(user, "French") == 0){
        printf("Bonjour");
    }
}
```

## 🚀 How to Run

1. Install GCC compiler (MinGW)
2. Compile:

   ```
   gcc filename.c -o output
   ```
3. Run:

   ```
   ./output
   ```
